# Snake Game con React (sin build tools)

## Descripción
Implementación del clásico juego Snake utilizando React vía CDN y Babel directamente en el navegador.

## Cómo ejecutar
1. Abrir el archivo `index.html`
2. Usar las flechas del teclado para mover la serpiente
3. Comer la comida roja para crecer y aumentar puntaje

## Reglas
- Si la serpiente choca con los bordes → Game Over
- Si choca consigo misma → Game Over
- Cada comida aumenta el puntaje

## Controles
- Flechas del teclado para mover serpiente
- Botón reiniciar tras perder

## Tecnologías
- React (CDN)
- Babel standalone
- JavaScript (ES6)
